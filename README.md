MeowAI, by 喵哥

喵哥AI，一个集成了gemini ,grok-beta ，flux 的绘画，人工智能聊天助手 （注意，前期不对中国大陆开放，因为API 的调用，都是用的官方接口，后期考虑更换

gemini的模型是 gemini-2.0-flash，支持视觉识别，flux的模型，绘画有可能会比较慢，还请耐心等待即可

你可以直接通过vercel 部署本项目
